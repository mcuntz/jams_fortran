The .f90 files can be used as is.
If you want to access the help, just double click on NR9F206H.HLP.
The file will be opened with winhelp.exe, which is the 16-bit version of winhlp32.exe.
Alternatively, copy all files to your hard drive, and edit the properties 
of the 'NR Fortran 90' shortcut so that it reflects the location of the NR9F206H.HLP 
file.  You can then start the help using the shortcut.
